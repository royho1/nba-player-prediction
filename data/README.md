# Data

## Included

- train5final.csv - Cleaned dataset of 601 NBA players who played 5 consecutive seasons post-draft.

## Optional

To re-run the data cleaning pipeline from scratch, download all_seasons.csv from Kaggle (https://www.kaggle.com/datasets/justinas/nba-players-data) and place it in this directory, then run: python src/data_cleaning.py
